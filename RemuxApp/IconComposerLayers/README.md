# Remux App Icon

`remux-app-icon-polished-master.svg` is the editable source for the Remux app
icon. The production app icon package lives at `../Remux.icon`.

Keep the pane geometry aligned to this grid:

- Column gutter: `39px`
- Row gutter: `39px`
- Left column right edge: `502`
- Right column left edge: `541`
- Top row bottom edge: `529`
- Bottom row top edge: `568`

Visual constraints:

- Dark terminal field.
- Oxide glass pane structure.
- Lifted teal active pane.
- Dark inset cursor.
- Amber live-signal accent.
- No decorative texture, baked Home Screen rim, or top highlight line.
- Clear Home Screen read at 40-64px.
